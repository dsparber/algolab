# Algorithms Lab 2020

## Build

```shell script
mkdir build
cd build
cmake ..
make -j8
```

- LP: Linear Programming
- MC: Min. Cut
- MF: Max. Flow
- MST: Minimum Spanning Tree
- SPShortest Path
- SW: Sliding Window
- UF: Union Find
