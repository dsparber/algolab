# Algorithms Lab 2020

## Build and run

```shell script
mkdir build
cd build
cmake ..
make -j8
```

### Legend
- 2C: 2-Coloring / Bipartition

- DP: Dynamic Programming
- DT: Delaunay Triangulation
- Geo: Geometric
- LP: Linear Programming
- MC: Min. Cut
- MF: Max. Flow
- MST: Minimum Spanning Tree
- SPShortest Path
- SW: Sliding Window
- UF: Union Find
